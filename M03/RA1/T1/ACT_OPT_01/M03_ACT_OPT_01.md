# EJERCICIO 1

## Definición de programa informático

Un programa informático es un conjunto de instrucciones escritas por un programador utilizando un lenguaje de programación, que una computadora ejecuta para resolver un problema o realizar una tarea específica.

## Componentes clave

- **Instrucciones:** Órdenes que indican qué debe hacer la computadora  
- **Lenguaje de programación:** Reglas y sintaxis que permiten escribir dichas instrucciones  
- **Ejecución:** Proceso mediante el cual la computadora interpreta y ejecuta las instrucciones

# EJERCICIO 2

## Comparativa Scrum vs Kanban

### Scrum
- Trabajo dividido en ciclos llamados **Sprints** (2–4 semanas)
- Flujo iterativo e incremental
- Roles definidos:
  - Product Owner
  - Scrum Master
  - Equipo de Desarrollo
- Objetivo: entregar incrementos funcionales al final de cada sprint

### Kanban
- Flujo de trabajo **continuo**
- Uso de un **tablero visual** (To Do, In Progress, Done)
- Limitación del trabajo en curso (**WIP limits**)
- Enfoque en detectar cuellos de botella y optimizar flujo

# EJERCICIO 3

## Lenguajes de bajo nivel vs alto nivel

### Bajo nivel
- Cercanos al lenguaje máquina
- Control directo del hardware
- Ejemplo:
  - Ensamblador
  - Código máquina

### Alto nivel
- Mayor abstracción
- Sintaxis más cercana al lenguaje humano
- Ejemplo:
  - Python
  - C++
  - Java

## Aplicación en IA

- **Uso principal:** lenguajes de alto nivel (Python)
  - Librerías como TensorFlow
  - Mayor productividad

- **Uso complementario:** lenguajes más cercanos al hardware (C++)
  - Optimización de rendimiento
  - Procesamiento crítico

# EJERCICIO 4

## Tipos de aprendizaje en IA

### 1. Aprendizaje Supervisado
- Usa datos etiquetados
- Aprende relación entrada → salida

**Ejemplo:**
- Clasificación de imágenes (gato vs perro)
- Predicción de precios

---

### 2. Aprendizaje No Supervisado
- No hay etiquetas
- Detecta patrones ocultos

**Ejemplo:**
- Clustering de usuarios
- Segmentación de clientes

---

### 3. Aprendizaje por Refuerzo
- Aprende mediante prueba y error
- Sistema de recompensas y penalizaciones

**Ejemplo:**
- Agente que aprende a jugar un videojuego
